Ionic Flat Colors

Why this has been created?

I created this project to help devs that like me are not so good with design.

Ionic is a great framework to develop hybrid apps, it already came with some colors by default, but sometimes you need to customize the colors and flat colors are very good to be used in apps.

Every time I start an app with Ionic I search on internet some flat color palettes to use and after some searchs I found an website of a designer called "vbaimas", by the way I want to thanks him for his work, this project is all based on his flat colors website called "http://www.flatuicolorpicker.com/". Thanks Vbaimas for you work.

So what is Ionic Flat Colors?

The Ionic Flat Colors is an expansion for the Ionic components. This IFC (Ionic Flat Colors) give new option of colors for Ionic components, flat colors that you can use and customize your app.

How can I start to use it?

To use Ionic Flat Colors is very simple, you just need to download the distribution version of the lib and add it in your project after the ionic css in your index file. After that you can use all of the classes with the flat colors.

I already have a project, does it mess up the ionic classes I alredy using?

No, this project doesn't change the Ionic default colors, it just add new options for you
