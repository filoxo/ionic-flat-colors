Why this project has been created?

I created this project to help developers that, like me, are not very good with design and colors.

Ionic is a great framework to develop hybrid apps. It already comes with some colors to customize your app, but sometimes you need more options.

Every time I start an app with Ionic I always search on internet for some flat color palettes to use and after some searches I always find myself on a website by a designer called "vbaimas". I'd like to thank him for his work as this project is based on his flat colors website called Flat Ui color Picker. Thanks Vbaimas for you work.

So what is the Ionic Flat Colors project?

The Ionic Flat Colors is an expansion for the Ionic components. The library gives you new options of colors for Ionic components to customize your app.

How can I get started?

Getting started with Ionic Flat Colors is very simple, you just need to download the distribution version of the library and add it to your project after the ionic css in your index file. After that you can use all of the classes of the project with the Ionic components.

I have an Ionic project and I want to use the Ionic Flat Colors library. Will the library break my css code?

No, this project does not change the Ionic default colors, it just adds new options for you
